Assignments

Checkpoints (20 points)

  1. Top Five Title Sequences (5 points):
    Visit Art of the Title and spend some time browsing the database of title sequences. Select five of your favorite title sequences and write a short description for each one outlining the characteristics you like about it. Create a new post on your class tumblr account with the links to the Art of the Title pages as well as your descriptions.

  2. After Effects Tutorials (5 points):

  3. Processing Tutorials (5 points):
    Go through the Processing Text Tutorials: Processing Overview, Coordinate System and Shapes, Color, and Typography.

  4. Nature of Code Ch. 1-4 (5 points):
    Visit the Nature of Code web site and read the Introduction and first four chapters: Vectors, Forces, Oscillation, and Particle Systems.

Projects (80 points)

  1. Sol Lewitt Animated Wall Drawing (20 points):
    Create an animated composition following the rules presented. The rules must be followed, but any room for interpretation is encouraged to be played with. The animation must be exactly 6 seconds long and must be able to loop seamlessly.

    Examples:
    • Sol LeWitt Animation

  2. 16 Bars (20 points):
    Use 8, 16, or 20 lyrical measures of a song (ie: a verse) to create a short video using kinetic typography animated in sync with the music. Use After Effects and the various text, shape and effects tools to implement the appropriate styles and animations to match the style of music and content of the lyrics.

    Examples:
    • Beauty Hip-Hop Kinetic Typography
    • Eminem - Rap God
    • Zach Rap

  3. Generative Composition (20 points):
    Using Processing create a generative animated composition. Your software should make use of variables, loops, objects, and random/noise functions.

    Examples:
    • Casey Reas - Process 10
    • Casey Reas - Process 13
    • Jared Tarbell - Substrate
    • Robert Hodgin - Magnetosphere
    • Robert Hodgin - Addition/Subtraction

  4. Interactive Graphics (20 points):
    Use mouse/touch interaction to affect on screen graphics through responsive animation. The software should be able to export as Javascript and run in the web browser.

    Examples:
    • Jono Brandel - Patatap
    • datadreamer - dailies
    • Hakim - Particles
